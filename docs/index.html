Rapid.js

Fast, lightweight, easy, flexible rendering engine

npm i rapid-render

Rapid.js is a lightweight WebGL rendering engine focused on 2D graphics. It provides a simple and intuitive API for developers to quickly build high-performance 2D games. Let's create something amazing together! ✨

Features

  • Fast Rendering
  • TileMap - YSort, isometric
  • Graphics Drawing
  • Text Rendering
  • Line Drawing
  • Custom Shaders
  • Mask
  • Frame Buffer Object
